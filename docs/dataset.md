# Trustpilot Dataset

## CSV Source
The canonical downloadable CSV is hosted on Kaggle:

- https://www.kaggle.com/datasets/ismetsemedov/trustpilot

Use this source for the latest/portable dataset file instead of committing large local CSV artifacts.

## Local File Policy

- Local scrape output path: `data/data.csv`
- `data/data.csv` is gitignored because the file can exceed 100 MB.
